The Position Property

First rule of position - do not abuse position!

  • position: static
  • position: relative
  • position: absolute
  • position: fixed
  • position: sticky
This is position sticky

Static is the default value of position.

Top, left, bottom, right have no effect.

Static

Relative positions the element relative to its normal position.

Top, left, bottom, right have an effect.

Relative

Absolute positions the element relative to its closest positioned ancestor.

Top, left, bottom, right have an effect.

An absolutely positioned element MUST have a relatively positioned parent.

Fixed positions the element relative to the browser window.

Cookie notices are an excellent use case for position: fixed.
